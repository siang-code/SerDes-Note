## 高速前端頻寬擴展技術：$f_T$ Doubler 與 Miller Cap Cancellation

### 數學推導
本頁筆記介紹了兩種在 Limiting Amplifier (LA) 或高頻放大器中用來提升頻寬（減小等效輸入電容）的經典電路技巧。

**1. Method V: $f_T$ Doubler (輸入電容減半技術)**
*   **電路動作**：將傳統差動對拆分為兩組 (M1/M2 與 M3/M4)。$V_{in}^+$ 僅驅動 M1，$V_{in}^-$ 僅驅動 M4，而 M2、M3 的閘極接至直流偏壓 $V_b$ (小訊號 AC ground)。
*   **電容減半推導**：
    1.  考慮左側差動對 (M1, M2)，輸入訊號為 $V_{in}^+$。兩者源極相連節點電壓設為 $V_{s1}$。
    2.  因為 M2 閘極為 AC ground，M1, M2 形成一個源極隨耦器與共閘極的結合。源極電壓會跟隨輸入：$V_{s1} = \frac{g_{m1}}{g_{m1}+g_{m2}} V_{in}^+$。假設電晶體匹配 ($g_{m1} = g_{m2}$)，則 $V_{s1} = \frac{1}{2} V_{in}^+$。
    3.  從 $V_{in}^+$ 看入，跨在 M1 的 $C_{gs1}$ 兩端的電壓差為 $(V_{in}^+ - V_{s1}) = \frac{1}{2} V_{in}^+$。
    4.  輸入電荷 $Q = C_{gs1} \times \frac{1}{2} V_{in}^+$，故**等效輸入電容 $C_{in(eff)} = 0.5 C_{gs1}$**。
*   **增益推導**：
    1.  M1 的小訊號汲極電流 $i_{d1} = g_m (V_{in}^+ - V_{s1}) = g_m (\frac{1}{2} V_{in}^+)$。
    2.  右側差動對 M3 由 $V_b$ 驅動，M4 由 $V_{in}^-$ 驅動，$V_{s2} = \frac{1}{2} V_{in}^-$。M3 的電流 $i_{d3} = g_m (0 - V_{s2}) = -g_m (\frac{1}{2} V_{in}^-)$。
    3.  兩者電流在左側負載 $R_D$ 匯合：$V_{out}^- = - (i_{d1} + i_{d3}) R_D = - \frac{1}{2} g_m R_D (V_{in}^+ - V_{in}^-)$。
    4.  同理 $V_{out}^+$ 亦有對稱貢獻，最終差動增益 $A_v = \frac{V_{out}^+ - V_{out}^-}{V_{in}^+ - V_{in}^-} = g_m R_D$。
    5.  **結論**：增益與傳統差動對相同，但輸入電容減半，因此頻寬極限 ($f_T$) 大幅提升。但代價是紅色筆記所寫的：**Double power dissipation** (需兩組 $I_{SS}$) 與 **Double IR drop** ($R_D$ 上流過雙倍 DC 電流)。

**2. Method VI: Miller Cap Cancellation (米勒電容抵消/中和技術)**
*   **問題**：傳統差動對中，閘-汲極電容 $C_{gd1}$ 因米勒效應，等效輸入電容會被放大為 $C_{gd1}(1+A_v)$，嚴重拖累高頻響應。
*   **解法**：加入交叉耦合電容 $C_{eq}$ (Cross-coupled capacitors)。
*   **抵消條件推導**：
    1.  輸入節點 $V_{in}^+$ 需提供充放電電流給兩個路徑：$C_{gd1}$ 與 $C_{eq}$。
    2.  流經 $C_{gd1}$ 的電流：$i_{Cgd1} = s C_{gd1} (V_{in}^+ - V_{out}^-)$。因為反相放大 $V_{out}^- = -A_v V_{in}^+$，所以 $i_{Cgd1} = s C_{gd1} (1 + A_v) V_{in}^+$。(此為正的等效大電容)
    3.  流經 $C_{eq}$ 的電流：$i_{Ceq} = s C_{eq} (V_{in}^+ - V_{out}^+)$。因為同相放大 $V_{out}^+ = +A_v V_{in}^+$，所以 $i_{Ceq} = s C_{eq} (1 - A_v) V_{in}^+$。(當 $A_v > 1$ 時，此項為負值，即**負電容**效應)
    4.  令總輸入電流 $i_{in} = i_{Cgd1} + i_{Ceq} = 0$ 以完全抵消米勒效應：
        $$C_{gd1}(1 + A_v) + C_{eq}(1 - A_v) = 0$$
        $$C_{eq} = C_{gd1} \frac{A_v + 1}{A_v - 1}$$
    5.  代入 $A_v = g_m R_D$，得到筆記上的公式：$C_{eq} \cong \frac{g_m R_D + 1}{g_m R_D - 1} C_{gd1}$。

### 單位解析
**公式單位消去：**
*   **差動增益公式**：$A_v = g_m \cdot R_D$
    *   $g_m$ 轉導：[A/V]
    *   $R_D$ 電阻：[V/A] (即歐姆 $\Omega$)
    *   $A_v = [\text{A/V}] \times [\text{V/A}] = [1]$ (無因次量，代表電壓放大倍率)
*   **米勒電容抵消公式**：$C_{eq} = C_{gd1} \frac{g_m R_D + 1}{g_m R_D - 1}$
    *   $C_{gd1}$ 電容：[F] (法拉)
    *   括號內分子分母皆為增益 $+/- 1$：$[1] / [1] = [1]$ (無因次量)
    *   $C_{eq} = [\text{F}] \times [1] = [\text{F}]$ (單位正確，為法拉)
*   **電容電流公式**：$i = s \cdot C \cdot V$
    *   $s$ 複頻率 ($j\omega$)：[1/s]
    *   $C$ 電容：[F] = [C/V] = [A·s/V] (因 $I = dQ/dt \Rightarrow \text{A} = \text{C/s}$)
    *   $V$ 電壓：[V]
    *   $i = [1/\text{s}] \times [\text{A}\cdot\text{s/V}] \times [\text{V}] = [\text{A}]$ (安培)

**圖表單位推斷：**
*   本頁無座標軸圖表，皆為電路架構示意圖 (Schematics)。

### 白話物理意義
*   **$f_T$ Doubler**：把一組差動對拆成兩組來分攤輸入訊號，藉由「源極電壓會跟著閘極動」的特性，讓跨在寄生電容上的壓差減半，成功騙過電路讓它覺得輸入電容變小了。
*   **Miller Cap Cancellation**：利用差動電壓「一邊下降、另一邊必定上升」的特性，外掛一顆交叉電容去產生「負電容」來抵消原本寄生電容的充電需求。

### 生活化比喻
*   **$f_T$ Doubler**：為了解決跑車（高頻寬需求）起步慢的問題，原本是一顆引擎拖整個車身（一個差動對吃所有 $C_{gs}$）。現在改裝成雙引擎（兩個差動對），雖然超級耗油（Double power），但引擎負擔減半，起步加速直接翻倍！
*   **Miller Cap Cancellation**：你的背包裡裝了一顆很重的石頭（米勒電容），讓你爬山很累（高頻衰減）。現在你在背包上綁了一大把會往上拉的氦氣球（交叉耦合電容產生的負電容），氣球的浮力剛好抵消石頭的重量，讓你感覺背包變輕了。

### 面試必考點
1.  **問題：$f_T$ Doubler 既然可以讓輸入電容減半，為什麼不每個 stage 都用？**
    *   **答案**：因為代價太高！它需要兩倍的尾電流 (Double power dissipation)，並且會讓負載電阻 $R_D$ 上的直流壓降變為兩倍 (Double IR drop)。在先進製程 (如 5nm/3nm) VDD 很低的情況下，過大的 IR drop 會吃掉 Voltage Headroom，導致電晶體無法維持在飽和區。通常只用在極度渴望頻寬的關鍵節點（如 TIA 或 LA 的第一級）。
2.  **問題：在 Miller Cap Cancellation 電路中，如果佈局 (Layout) 沒畫好，導致 $C_{eq}$ 比 $C_{gd1}$ 大太多，會發生什麼事？**
    *   **答案**：會產生過大的「負電容」效應 (Over-neutralization)。當輸入端看到的總電容變成負值時，電路會產生右半平面極點 (RHP Pole)，導致系統不穩定，頻率響應出現 Peaking，嚴重時甚至會直接起振變成振盪器 (Oscillator)。
3.  **問題：筆記中提到一般差動對的 Source 端是 "小訊號 AC ground"，這在物理上代表什麼？對輸入電容有何影響？**
    *   **答案**：因為差動訊號一正一負 ($V_{in}/2$ 與 $-V_{in}/2$)，兩顆電晶體的電流變化量 $+i$ 與 $-i$ 在共同源極 (Common Source) 互相抵消，導致該點電壓不會跳動，形成虛擬地 (AC ground)。這使得輸入訊號必須對完整的 $C_{gs}$ 充放電（因為另一端死死釘在地上），所以看到了最大的寄生電容。

**記憶口訣：**
- $f_T$ 翻倍法：**功耗換速度，電壓空間少一半。**
- 米勒抵消法：**交叉接電容，一正抵一負。** (小心過度補償變振盪)
