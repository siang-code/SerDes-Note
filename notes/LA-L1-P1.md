## [Limiting Amplifiers (限幅放大器) 與串接頻寬推導]

### 數學推導
1. **單級轉移函數 (Single-Stage Transfer Function)**
   一個帶有阻抗負載 ($R_D, C_L$) 的差動對放大器，其直流增益為 $|A_0| = g_{m1,2} R_D$，主極點頻率為 $\omega_0 = (R_D C_L)^{-1}$。
   其單級的轉移函數可表示為：
   $H(s) = \frac{A_0}{1 + \frac{s}{\omega_0}}$

2. **多級串接轉移函數 (N-Stage Cascaded Transfer Function)**
   將 $n$ 級相同的放大器串接，假設後級不會產生額外的複雜負載效應（或者等效電容皆已併入 $C_L$ 中），且雖然訊號逐級放大，但為了計算頻寬仍使用小訊號分析。總轉移函數為單級的 $n$ 次方：
   $H_{tot}(s) = \left( \frac{A_0}{1 + \frac{s}{\omega_0}} \right)^n$

3. **總體 -3dB 頻寬 (Total -3dB Bandwidth)**
   定義總頻寬 $\omega_{-3dB}$ 為總增益下降至「總直流增益的 $1/\sqrt{2}$ 倍（即下降 3dB）」時的頻率：
   $|H_{tot}(j\omega_{-3dB})| = \frac{|A_0|^n}{\sqrt{2}}$
   將 $s = j\omega_{-3dB}$ 代入總轉移函數取絕對值：
   $\left| \frac{A_0}{1 + j\frac{\omega_{-3dB}}{\omega_0}} \right|^n = \frac{|A_0|^n}{\sqrt{2}}$
   $\left( \frac{1}{\sqrt{1 + (\frac{\omega_{-3dB}}{\omega_0})^2}} \right)^n = \frac{1}{\sqrt{2}}$
   兩邊取平方並倒數：
   $\left(1 + \left(\frac{\omega_{-3dB}}{\omega_0}\right)^2\right)^n = 2$
   $1 + \left(\frac{\omega_{-3dB}}{\omega_0}\right)^2 = 2^{1/n}$
   $\omega_{-3dB} = \omega_0 \sqrt{2^{1/n} - 1}$

4. **帶寬縮減近似公式 (Bandwidth Shrinkage Approximation)**
   當級數 $n$ 夠大時，利用泰勒展開式近似，可得頻寬縮減因子（Bandwidth Shrinkage Factor）大約為 $\frac{\sqrt{\ln 2}}{\sqrt{n}} \approx \frac{0.83}{\sqrt{n}}$。
   筆記中簡化為：$BW_{tot} \approx \omega_0 \frac{0.8}{\sqrt{n}}$

### 單位解析
**公式單位消去：**
1. **增益公式**：$|A_0| = g_{m} \times R_D$
   $[A/V] \times [\Omega] = [A/V] \times [V/A] = [無單位] (V/V)$，增益為純量。
2. **主極點公式**：$\omega_0 = (R_D C_L)^{-1}$
   $([\Omega] \times [F])^{-1} = ([V/A] \times [C/V])^{-1} = ([C/A])^{-1} = ([s])^{-1} = [rad/s]$ 或 $[Hz]$，物理意義為頻率。
3. **總體頻寬公式**：$\omega_{-3dB} = \omega_0 \times \sqrt{2^{1/n} - 1}$
   $[rad/s] \times [無單位] = [rad/s]$，物理單位與原本單級頻寬相同。

**圖表單位推斷：**
📈 圖表單位推斷：串接級數對增益與頻寬的影響
- **X 軸**：串接級數 (n stages) [無單位]，範圍 1~6 級。
- **左 Y 軸 (藍線)**：直流總增益 (dc gain) [dB]，範圍 10~60 dB（呈線性上升，因為是 dB 相加）。
- **右 Y 軸 (紅線)**：總體頻寬 (BWtot) [GHz]，範圍通常在數十 GHz 內，呈根號反比下降曲線。

### 白話物理意義
將放大器一級一級串接，雖然總增益會以倍數暴增，但每一次通過放大器都會被寄生 RC 延遲「砍」一次高頻，導致總頻寬會以 $\frac{1}{\sqrt{n}}$ 的速度漸漸縮水。

### 生活化比喻
就像「傳話遊戲」，每多加一個人（一級放大器）接力，聲音雖然可以被傳遞到很遠的地方（增益增加），但每個人的反應和口齒都會稍微拖慢一點速度（RC 寄生極點延遲）。傳遞的人數越多，能準確傳遞的「講話語速」（頻寬）就必須越慢，否則後面的人就會聽不懂糊成一團（高頻衰減）。

### 面試必考點
1. **問題：在設計 Limiting Amplifier 時，為何必須採用多級串接 (Cascading) 架構？**
   - 答案：因為高速 SerDes 接收端前端資料訊號微弱，需要極大的總增益（如 40~60dB）且同時必須維持極寬的頻寬（數 GHz 以上）。受限於給定製程的 Gain-Bandwidth Product (GBW) 限制，單級放大器無法同時達成這兩個嚴苛條件，必須利用多級串接來分散增益壓力。
2. **問題：多級串接的總頻寬為何會隨級數增加而下降？下降的比例因子公式為何？**
   - 答案：每增加一級放大器，訊號就會多經過一個極點，導致高頻響應的衰減不斷疊加。總頻寬下降的精確比例因子為 $\sqrt{2^{1/n} - 1}$；當級數 $n$ 較大時，頻寬下降的速度約正比於 $\frac{1}{\sqrt{n}}$。
3. **問題：圖表中「有限頻寬減少換取總增益上升」的設計哲學是什麼？**
   - 答案：在對數尺度 (dB) 下，總增益隨串接級數 $n$ 呈「線性」暴增（例如一級 10dB，六級就 60dB）；但總頻寬卻僅以 $\frac{1}{\sqrt{n}}$ 的速度緩慢下降。增益上升的速度遠大於頻寬下降的速度，因此這是一個極具效益的 Trade-off，是實現寬頻高增益放大的唯一解。

**記憶口訣：**
串接增益次方跳，頻寬根號 n 衰掉，增益換頻寬最有效！
