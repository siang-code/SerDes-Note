## 一階 Sigma-Delta Modulator (1-order ΣΔ Mod) 於小數分頻 PLL 的應用

### 數學推導
由系統方塊圖（Block Diagram）可逐步推導出 Z 域的系統方程式：
1. **誤差計算**：積分器輸入端為目標訊號 $X(z)$ 減去延遲一拍的輸出 $z^{-1}Y(z)$：
   $$E(z) = X(z) - z^{-1}Y(z)$$
2. **積分器 (Integrator)**：積分器的轉移函數為 $\frac{1}{1-z^{-1}}$，其輸出 $W(z)$ 為：
   $$W(z) = E(z) \cdot \frac{1}{1-z^{-1}} = \frac{X(z) - z^{-1}Y(z)}{1-z^{-1}}$$
3. **量化器 (Quantizer)**：量化過程等效於在訊號上疊加一個量化雜訊 $Q(z)$，因此輸出 $Y(z)$ 為：
   $$Y(z) = W(z) + Q(z)$$
4. **代入化簡**：將 (2) 代入 (3) 中：
   $$Y(z) = \frac{X(z) - z^{-1}Y(z)}{1-z^{-1}} + Q(z)$$
5. **消除分母**：等式兩邊同乘 $(1-z^{-1})$：
   $$(1-z^{-1})Y(z) = X(z) - z^{-1}Y(z) + (1-z^{-1})Q(z)$$
6. **展開與移項**：
   $$Y(z) - z^{-1}Y(z) = X(z) - z^{-1}Y(z) + (1-z^{-1})Q(z)$$
   兩邊同時消去 $-z^{-1}Y(z)$，可得最終輸出方程式：
   $$Y(z) = X(z) + (1-z^{-1})Q(z)$$

由上式可定義出兩個極為重要的轉移函數：
- **信號轉移函數 (STF, Signal Transfer Function)**：
  令 $Q(z) = 0$ 則 $STF = \frac{Y(z)}{X(z)} = 1$
  *(代表輸入的小數分頻比訊號能無失真地通過。)*
- **雜訊轉移函數 (NTF, Noise Transfer Function)**：
  令 $X(z) = 0$ 則 $NTF = \frac{Y(z)}{Q(z)} = 1 - z^{-1}$
  *(這是一個一階高通濾波器，實現了「雜訊整形 Noise Shaping」的核心概念。)*

### 單位解析
**公式單位消去：**
在純數位的 Sigma-Delta Modulator 運算中，所有變數皆為數位數值或純比例，無實體物理單位：
- $X(z)$：輸入的小數分頻比 $\alpha$，[無因次] (Dimensionless)
- $Y(z)$：輸出給除頻器的控制訊號 (通常為整數 0 或 1)，[無因次]
- $Q(z)$：量化器的捨去/進位誤差，[無因次]
- $z^{-1}$：單位時間延遲，[無因次]

將單位代入最終公式 $Y(z) = X(z) + (1-z^{-1})Q(z)$：
$[無因次] = [無因次] + ([無因次] - [無因次]) \times [無因次]$
等式兩邊單位完全吻合，皆為無因次。

**圖表單位推斷：**
📈 圖表單位推斷：
筆記右上角為兩個時域模擬運算表格（欄位對應離散時間 step $n$）：
- $x$ 欄（輸入）：目標小數比 [無因次]，典型範圍 $0 \sim 1$（如表中的 0.2 = 1/5，或 0.38 = 19/50）。
- $w$ 欄（積分狀態）：積分器累積值 [無因次]，典型範圍 $-1 \sim 2$。
- $y$ 欄（輸出）：量化器輸出 [無因次]，此處為 1-bit 量化器，僅有 0 或 1 兩種狀態。
- $q$ 欄（量化誤差）：定義為 $w - y$ [無因次]，典型範圍 $-0.5 \sim 0.5$。

### 白話物理意義
一階 Delta-Sigma Modulator 就是一個「記恨的平均機器」：它為了用粗糙的整數（0和1）來湊出精準的小數（例如0.2），會把每次四捨五入所產生的誤差用「積分器」記在帳上，留到下一次修正。這樣長期平均下來數字絕對準確，同時把短時間內忽大忽小的湊數誤差（量化雜訊）全部推擠到高頻去。

### 生活化比喻
假設你要跟室友分攤網飛訂閱費，你每天應付 0.2 元（$X=0.2$），但世界上最小的硬幣是 1 元（量化器 $Y$ 只能出 0 或 1）。
所以你拿個帳本（積分器 $W$）記帳：
第一天你欠 0.2 元，不到 0.5 元就不給錢（$Y=0$），把欠的 0.2 記到明天。
到了第三天，你累計欠了 0.6 元，超過 0.5 了，你終於掏出 1 元硬幣給室友（$Y=1$）。
給完 1 元後，你反而多付了 0.4 元，所以帳本變成 -0.4 元（這就是誤差回饋 $Q$）。
這樣持續下去，你每 5 天會給出一次 1 元，長期來看，你每天真的剛好只付了平均 0.2 元！

### 面試必考點
1. **問題：請寫出一階 SDM 的 NTF (Noise Transfer Function)，並解釋它在 Fractional-N PLL 裡面發揮什麼作用？**
   → **答案：** $NTF = 1 - z^{-1}$。在頻域上這是一個高通濾波器。它的作用是「雜訊整形 (Noise Shaping)」，將量化器產生的白雜訊推擠到高頻。在 Frac-N PLL 系統中，這些高頻的量化雜訊後續會自然地被具有低通特性的 Loop Filter 給濾除，從而保證了壓控振盪器 (VCO) 輸出的頻譜純淨度。
2. **問題：根據筆記，當輸入 $X$ 分別是 0.2 (1/5) 和 0.38 (19/50) 時，一階 SDM 會有什麼致命的非理想特性？**
   → **答案：** 會產生週期性的極限環振盪（Limit Cycle 或 Idle Tones）。因為 Fractional-N PLL 的分頻比通常是簡單的有理數（Rational number），在一階 SDM 中會導致輸出的 $Y[n]$ 序列呈現死板的週期重複（如 0.2 會每 5 週期重複一次）。這種規律的週期性會在輸出的相位雜訊頻譜上產生明顯的離散突波（Fractional Spurs）。
3. **問題：為了解決一階 SDM 產生 Spurs 的問題，通常會採用什麼架構？**
   → **答案：** 需要引入更高的隨機性 (Randomness)。業界標準做法是使用高階的 Sigma-Delta Modulator，特別是多階級聯的 MASH (Multi-Stage Noise Shaping) 架構（例如 MASH 1-1-1）。高階架構不僅能提供更陡峭的雜訊整形斜率（將雜訊推向更高頻），還能徹底打亂序列的週期性，將能量打散成類似白雜訊的基底，從而消除 Spurs。

**記憶口訣：**
一階SDM：訊號 STF 等於 1，雜訊 NTF 減 $z^{-1}$。
小數湊整數，誤差往後推；遇到週期生 Spurs，MASH 高階來打散。
