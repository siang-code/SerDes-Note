## Wide Range Modulus Divider 與 50% Duty Cycle 除頻器

### 數學推導
這份筆記涵蓋了高速頻率合成器 (Frequency Synthesizer) 中極為關鍵的兩個模塊：可程式化多模除頻器 (Multi-Modulus Divider, MMD) 以及具有精準 50% Duty Cycle 的除頻器。

**1. Multi-Modulus Divider (MMD) 總除頻比推導：**
- MMD 是由 $N$ 個 $\div 2 / 3$ cell 串聯而成。每一個 cell 都有一個控制訊號 $P_i$（其中 $i = 0 \sim N-1$）。
- **運作規則**：
  - 當 $P_i = 0$ 時，該級 cell 單純進行 $\div 2$。
  - 當 $P_i = 1$ 時，該級 cell 在一個完整的 MMD 輸出週期內，會進行**一次 $\div 3$**，其餘時間皆進行 $\div 2$。
- **權重推導**：
  - 第 0 級（最前端）直接吃 $ck_{in}$，如果它進行一次 $\div 3$，相當於多吞噬了 $1$ 個 $ck_{in}$ 週期，權重為 $2^0 = 1$。
  - 第 1 級吃的是第 0 級的輸出（頻率約為一半），如果第 1 級進行一次 $\div 3$，相當於在它的輸入多吞了一週，換算回最前端的 $ck_{in}$ 就是多吞了 $2$ 週期，權重為 $2^1 = 2$。
  - 依此類推，第 $i$ 級若觸發 $\div 3$，會使總週期增加 $2^i$ 個 $ck_{in}$。
- **總除頻比方程式**：
  - 基礎除頻比（當所有 $P_i = 0$ 時，全部 $\div 2$）為 $2^N$。
  - $$Div\_Ratio = 2^N + P_0 \cdot 2^0 + P_1 \cdot 2^1 + P_2 \cdot 2^2 + \dots + P_{N-1} \cdot 2^{N-1}$$
  - 可程式化範圍：$2^N \sim 2^{N+1} - 1$。
- **圖中實例驗證**：
  - 假設 $N=3$，且 $P_0=1, P_1=0, P_2=1$。
  - $$Div\_Ratio = 2^3 + (1 \times 1) + (0 \times 2) + (1 \times 4) = 8 + 1 + 0 + 4 = 13$$
  - 波形圖完美吻合：1 個 $FO3$ 週期包含了剛好 13 個 $ck_{in}$ 週期。

**2. 50% Duty Cycle $\div 3$ 電路推導：**
- 圖中上半部由兩個正緣觸發 (Positive-edge triggered) 的 D-FF 與邏輯閘組成標準的 $\div 3$ 計數器。其輸出節點 $V_P$ 的波形為：High 1 個週期，Low 2 個週期。
  - $Duty\_Cycle_{VP} = \frac{1 T_{in}}{3 T_{in}} = 33.3\%$
- 圖中下半部增加了一個**負緣觸發 (Negative-edge triggered)** 的 D-FF。
  - 它的輸入吃 $V_P$，並在 $ck_{in}$ 的下降緣觸發，這在時間軸上相當於把 $V_P$ 的波形精準向右平移了半個時脈週期（$0.5 T_{in}$），產生訊號 $V_R$。
- 最後將 $V_P$ 與 $V_R$ 通過一個 OR Gate 疊加：
  - 疊加後的高電位時間 $T_{high} = 1 T_{in} \text{ (來自 } V_P) + 0.5 T_{in} \text{ (來自 } V_R \text{ 延遲的部分)} = 1.5 T_{in}$。
  - $$Duty\_Cycle_{out} = \frac{1.5 T_{in}}{3 T_{in}} = 50\%$$

### 單位解析
**公式單位消去：**
- **MMD 週期轉換公式**：$T_{out} = Div\_Ratio \times T_{in}$
  - $T_{out}$ [s]：輸出週期
  - $Div\_Ratio$ [UI/UI] = [純量]：除頻比例，無單位
  - $T_{in}$ [s]：輸入週期
  - **消去**：$[s] = [純量] \times [s]$，左右單位一致。
- **Duty Cycle 公式**：$Duty\_Cycle = \frac{T_{high}}{T_{period}} \times 100\%$
  - $T_{high}$ [s]：高電位持續時間
  - $T_{period}$ [s]：總週期時間
  - **消去**：$\frac{[s]}{[s]} \times [\%] = [純量] \times [\%] = [\%]$，單位一致。

**圖表單位推斷：**
📈 **MMD 除頻波形圖 (ckin, FO1, FO2, FO3)**
- **X 軸**：時間 [$UI_{in}$] 或 [$ps$]，典型範圍為 $0 \sim 20\ UI_{in}$（圖中展示了 13 個週期）。
- **Y 軸**：電壓 [$V$]，典型為 CMOS 邏輯準位（如 $0 \sim 0.9V$ 於 28nm 製程）。

📈 **50% Duty Cycle $\div 3$ 波形圖 (ckin, VP, VR, ckout)**
- **X 軸**：時間 [$UI_{in}$] 或 [$ps$]，典型範圍為 $0 \sim 3\ UI_{in}$。
- **Y 軸**：電壓 [$V$]，典型為 CMOS 邏輯準位。

### 白話物理意義
- **MMD**：就像一個可變速的齒輪箱，平常每個齒輪都是「轉2圈輸出1圈」，但只要收到指令（$P_i=1$），那個齒輪就會「稍微打滑一下，變成轉3圈才輸出1圈」。透過不同齒輪級別的打滑組合，就能精準控制引擎（VCO）要轉幾圈，車輪（輸出）才會轉一圈。
- **50% Duty Cycle $\div 3$**：原本的波形太瘦了（只有 33% 寬），所以我們利用時鐘的「反相邊緣」把波形往後複製半步，然後把原本的跟複製的「疊加（OR）」在一起，波形就剛好長胖到完美的一半（50%）了！

### 生活化比喻
- **MMD 架構**：想像跨國快遞轉運站。第 0 級是機車快遞（最高速），第 1 級是貨車，第 2 級是飛機。平常機車收 2 件貨就打包給貨車，貨車收 2 包就上飛機。如果老闆下令 $P_2=1$（飛機站要求多吞貨），這不代表機車只要多收 1 件，而是飛機站要多收 1 箱貨車的貨，換算回最前線的機車，機車小哥必須要多跑 $2^2 = 4$ 趟！這就是不同級別 $P_i$ 擁有不同權重（$2^i$）的原因。
- **50% Duty Cycle 產生器**：就像你烤了一片吐司，但果醬只塗了前面 1/3。你覺得這樣咬起來口感不均勻，於是你拿另一片吐司，把果醬塗在稍微往右平移半步的位置，然後兩片疊在一起（OR Gate），這樣整份三明治看起來就剛好有一半是塗滿果醬的了！

### 面試必考點
1. **問題：在高速 SerDes 中，為什麼經常要求 Divider 必須輸出精準的 50% Duty Cycle？如果只有 33% 會怎樣？**
   → **答案**：高速 SerDes 常使用 Half-rate 甚至 Quarter-rate 架構，這意味著 Phase Detector (PD) 或後級電路會「同時利用時脈的正緣與負緣」來取樣資料。如果 Duty cycle 偏離 50%（例如 33%），正緣到負緣的時間與負緣到正緣的時間會不對稱，這會嚴重壓縮 Setup/Hold time margin，並在系統中引入嚴重的 Deterministic Jitter (DJ)。
2. **問題：請比較 MMD (Multi-Modulus Divider) 與傳統 Pulse-Swallow Divider 架構，為何高速 PLL 偏好 MMD？**
   → **答案**：傳統 Pulse-Swallow 架構包含 Prescaler、Program Counter 與 Swallow Counter，其回授路徑（Feedback path）很長，導致 Critical path delay 很大，在高頻（如 >10GHz）下容易 setup time violation。而 MMD 採用模組化級聯，**只有第一級運作在最高頻**，後續頻率依序減半，且控制訊號的傳遞是區域性的（Local），沒有貫穿全局的長回授線，因此功耗更低、能達到的最高操作頻率也更高。
3. **問題：圖中這個巧妙的 50% duty $\div 3$ 電路，在實際下線 (Tape-out) 時最大的隱憂是什麼？**
   → **答案**：它極度依賴輸入時脈 ($ck_{in}$) 自身的 Duty Cycle。因為 $V_R$ 是由 $ck_{in}$ 的「負緣」觸發產生，如果 $ck_{in}$ 受到前端電路影響，Duty cycle 不是精準的 50%，那麼 $V_P$ 與 $V_R$ 之間的延遲就不會剛好是 $0.5 T_{in}$。延遲一錯，最後 OR 閘疊加出來的 $ck_{out}$ 也絕對不會是 50%，這種 DCD (Duty Cycle Distortion) 會沿著電路鏈一路惡化下去。

**記憶口訣：**
- **MMD**：「**2的N次方打底，P控吞嚥湊零頭，高速串聯無長線。**」 (說明 MMD 是 Base + Offset，且解決了長回授路徑問題)
- **50% Div3**：「**正緣生三分，負緣延半步，OR閘一疊變一半。**」 (完美描述 33% + 負緣延遲 + OR = 50% 的物理意義)
